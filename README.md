# FastAPI JWT Auth API

Proyecto educativo desarrollado con FastAPI para comprender autenticación basada en JWT.

## Tecnologías

- Python
- FastAPI
- Uvicorn
- Python-JOSE

## Funcionalidades

- Generación de JWT
- Endpoint de login
- Expiración de tokens

## Ejecutar

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```
